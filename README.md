# Subscription and Churn Analysis

## Tools Used :
* PostgreSQL (pgAdmin)
* SQL
* Microsoft Excel

## Description :
This project analyzes subscription data to understand customer behavior, churn patterns, and revenue trends. It focuses on identifying active vs churned users and evaluating business performance using key metrics.

## Key Analysis :
* Active vs churned users identification
* Churn rate calculation
* Monthly Recurring Revenue (MRR) analysis
* Revenue distribution by city
* Customer lifetime value insights
