Subscription & Churn Analysis

Tools Used :
* PostgreSQL (pgAdmin)
* SQL
* Microsoft Excel
  
Description :
This project analyzes subscription data to understand user churn, retention behavior, and revenue patterns.

Key Analysis :
* Active vs churned users
* Monthly Recurring Revenue (MRR)
* Churn rate calculation
* Users with increasing subscription fees
* Revenue by city
* Longest active subscribers
* Lifetime revenue ranking
* Reactivated users after churn
  
Files :
* subscription_and_churn_analytics.sql – SQL queries and analysis
* subscription_and_churn_analytics.xlsx – Excel dashboard
